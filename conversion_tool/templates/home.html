ALLAN: IMGT/HLA ALLele to UNOS ANtigen conversion tool

About

This web tool facilitates the mapping of molecular HLA nomenclature to UNOS serological equivalents required for entering donor's HLA type into UNet system. The tool obliges the guidelines laid by Organ Procurement and Transplant Network for antigen mapping. The tool supports antigen mapping for alleles from seven loci i.e. (A, B, C, DRB1, DQA1, DQB1, DRB3/4/5) for unambiguous HLA typing (allele, allele list) and six loci (A, B, C, DRB1, DQB1, DRB3/4/5) for ambiguous HLA typing (genotype list strings and multiple allele codes).
