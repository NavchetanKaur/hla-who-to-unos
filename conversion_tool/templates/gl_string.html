Genotype List String

Enter 3, 4, 5, 6 loci derived genotype list string in the prescribed format in the text box below, select the population/race and click on "Convert" to map the alleles to UNOS antigen and designated Bw4 or Bw6 epitopes. Alleles included in the string should be either medium resolved to specific HLA protein field or fully resolved. Protein expression specifying characters L, N, S, Q can be used but not the suffixes for G or P groups
